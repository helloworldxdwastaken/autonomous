Agent Architecture

Each agent has a specialty, personality, and unique set of skills

Reader Agent

GPT-4o-mini / Llama 3.1

Reads scientific papers, extracts key variables, claims, and methodologies. Indexes content semantically for instant search.

Reviewer Agent

GPT-5 / Gemini Pro

Evaluates scientific soundness, methodologies, and statistical validity of findings. Ensures academic rigor.

Contradictions Agent

GPT-5-Reasoning

Searches for inconsistencies between papers, questions conclusions, and generates scientific debates between contradictory findings.

Writer Agent

GPT-5 / Gemini

Drafts complete scientific articles, research reports, and technical documentation in academic format.

Simulation Agent

NVIDIA BioNeMo

Executes molecular simulations of protein-protein interactions using NVIDIA BioNeMo and ESMFold.

Translator Agent

Gemini-Pro

Translates papers between multiple languages (English, Spanish, Chinese, Russian) while maintaining technical precision.

Atlas Agent

Llama-4 / OpenAI Embeddings

Maps connections between concepts and papers in a navigable, interactive 3D knowledge graph.

Improvement Agent

GPT-5-Thinking

Evaluates the performance of other agents, optimizes prompts, and continuously improves reasoning strategies.

TeamLead (Orchestrator)

GPT-5-Reasoning / Gemini-1.5

Coordinates all agents, intelligently distributes tasks, and decides which processes to execute based on priorities and available resources.
